Instalacion de servicio local para sitio web

Publicado 18-10-2022

Sergio Armando Pérez Rodríguez
Ingeniero en sistemas computacionales

1.Instalar una paqueteria que te incluya un servidor web local (XAMPP)

Vamos a la pagina web de la paqueteria en el cual esta ocasion sera XAMPP
el cual te incluye el servidor web local de apache, ademas de otro servicios
como MariaDB y el lenguaje de programación de php.

2.Escoger la version que vamos a descargar, dependiendo de tu sistema operativo

Damos clic en la seccion de descargas y ahi nos apareceran diversas opciones a
escoger. Debemos escoger la version que queramos pero esocgiendo la cual se ajuste
con la version de nuestro sistema operativo.

3. Escogemos que servicios vamos a Instalar

En el transcurso de la instalación vamos a seleccionar todos los servicios que queramos que se instalen con el xampp.

4. Guardamos nuestro proyecto en el servidor

Una vez instalado el xampp, nos dirigimos a la siguiente ruta:

C:\xampp\htdocs

En esta parte donde vamos a poner la carpeta de nuestra carpeta. Para este ejemplo la carpeta es "CursoTPW".

Si entramos en la carpeta justamente veremos los directorios del curso.

5.Iniciamos nuestro servido web local

Entramos al panel de Xampp y le damos clic en "start" al servicio de apache.

6.Ingresamos a nuestro proyecto por la ruta donde guardamos la carpeta

En nustro navegador escribiremos la direccion de donde esta la ruta de nuestro
proyecto. En este ejemplo, nuestro proyecto esta ubicado en:

http://localhost/CursoTPW/Actividad-2-01/index.html
